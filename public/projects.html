CSIRO Sonification Module

Open-source software developed by me during my CSIRO summer studentship on the LOCUST system. The LOCUST system is a radio-telescope set designed to engage students in science classrooms with STEM and I developed a code module meant to be used to engage students who are not able to see the visual data of graphs and plots by sonification of the radio waves. This project included extensive speed optimisations and asynchronous programming due to the heavy computation of complex algorithums being run on a Raspberry-Pi.

Automatic Code Smell Detection

Python research software I helped develop for a research project at university. The code will take the syntactic tree from the C++ compiler and analyse it to identify various code smells which can roughly be defined as refactorable code, checking a condition to return explicite boolean values rather than returning the value of the condition checked and duplicate code inside if-else statements that can be moved before or after the if-else. More examples and documentation should be available at the fork of the submission repository I have linked.

Simple Game in Python

A simple game GUI I made in Python using the Pygame library, before learning Object Oriented Programming so it's mostly functional programming apart from a patch made using a Python object. There was a lot of refactoring and abstraction involved though to allow me to more easily create the GUI I developed layers of abstraction until it was comparable to HTML for me.

Matlab Calculator GUI Project

Developed a Matlab program for an introductory coding course which would compute, interpret and visualise the reduced row echelon form of a matrix representing a system of linear equations.

C++ Terminal Hangman

C++ coded game of hangman played in the terminal, uses a hard-coded matrix and displays progressivly more lines the more erros the player makes. Uses a psuedo-random number generator to determine the word being played from a dictionary of words also hard-coded into the game.
